# Lamplight Digital

The website for [Lamplight Digital](https://lamplightdigital.co.uk), a small
web studio in Leeds.

One page of plain HTML, CSS and a little JavaScript.

## What's here

| File | What it is |
| --- | --- |
| `index.html` | The whole site |
| `styles.css` | Styles, including the light and dark themes |
| `scripts.js` | Theme toggle and the footer year |
| `lamp-pixel-art.svg` | The lamp logo (navy, for light mode) |
| `lamp-pixel-art-dark.svg` | The lamp logo (cream, for dark mode) |
| `Abordage-Regular.woff2` | The heading font |
| `previews/` | Screenshots of the sites in the portfolio |
| `favicon.*`, `apple-touch-icon.png`, `og-image.png` | Icons and the link preview image |

## Updating the portfolio

Each project is an `<article class="project-card">` in `index.html`. Copy
one, swap the screenshot in `previews/`, and change the text and links.

## Deploying

It's a static site. Put the files on any web server.
